 Final Project - Biostatistics

  This repository contains the final project for a biostatistics course.

  ## Topic

  The project examines whether the severity of anxiety symptoms is associated with the severity of depressive symptoms among adults, and whether this association
  remains after adjustment for age, sex, self-rated health, and lifestyle-related factors.

  ## Data

  The analysis uses the 2022 National Health Interview Survey (NHIS) Sample Adult dataset.

  Main variables:

  - PHQ-8 depressive symptom score
  - GAD-7 anxiety symptom score
  - Age
  - Sex
  - Self-rated health
  - Smoking status
  - Alcohol consumption
  - Physical activity
  - BMI category
  - Educational level

  ## Planned Analysis

  The project includes:

  - Data cleaning and recoding
  - Calculation of PHQ-8 and GAD-7 scores
  - Missing data assessment
  - Multiple imputation diagnostics
  - Descriptive statistics using TableOne
  - Multiple linear regression

  ## Notes

  The analysis is observational and cross-sectional. Results are interpreted as associations, not causal effects.
